# 🌿 Huerto Feliz - Landing Page

## 📋 Descripción del Proyecto

Landing page profesional para **Huerto Feliz**, una empresa especializada en servicios de jardinería con más de 10 años de experiencia en el mercado chileno.

## 🎯 Características Principales

### ✨ Diseño y Funcionalidades
- **Diseño Responsivo**: Optimizado para dispositivos móviles, tablets y desktop
- **Interfaz Moderna**: Utiliza Bootstrap 5 con estilos personalizados
- **Animaciones Suaves**: Transiciones y efectos hover para mejor experiencia de usuario
- **Navegación Fluida**: Menú de navegación con scroll suave entre secciones

### 📱 Secciones del Sitio

1. **🏠 Hero Section**: Presentación impactante con llamada a la acción
2. **🌱 Servicios**: Carrusel interactivo con 4 servicios principales
3. **📜 Historia**: Trayectoria de la empresa con estadísticas y testimonios
4. **ℹ️ Sobre Nosotros**: Misión, valores, galería de proyectos y ubicación
5. **📧 Contacto**: Formulario completo con validación y información de contacto
6. **🔗 Footer**: Enlaces, redes sociales y datos de contacto

### 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica del sitio
- **CSS3**: Estilos personalizados con animaciones
- **Bootstrap 5**: Framework CSS para diseño responsivo
- **JavaScript**: Validación de formularios y interactividad básica

1. **Requisitos Previos**:

   - Conexión a internet para cargar Bootstrap desde CDN

2. **Ejecución**:
   - Abrir `index.html` en cualquier navegador web
   - O usar un servidor local (recomendado para desarrollo)
